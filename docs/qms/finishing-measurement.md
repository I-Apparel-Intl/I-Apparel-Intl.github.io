# 🔍 Finishing Measurement

**Measure and verify finished products before final approval.**

  <img src="/img/finishing-measurement.png" width="60%" alt="Finishing Measurement" />
</div>

---

## 📋 Description

This feature is used to measure and inspect finished products to ensure they meet quality standards before proceeding to the next stage.

---

## 🪜 Steps

### Step 1 — Scan Worker ID

> Before proceeding, the worker must be identified.

1. Tap the **📷 Camera icon** or **🔍 QR icon** at the top left to open the scanner.
2. **Scan the Worker QR Code** to load the worker's information.
3. Once scanned, the worker name and ID will appear (e.g., `NHANH SOKHAI - D010966`).

### Step 2 — Choose Line

> Select the production line before proceeding with measurement.

1. After the worker ID is loaded, a **dropdown field** will appear below the worker name.
2. Tap the dropdown and **select the Line number** (e.g., `02`).
3. The selected line will be confirmed and displayed in the field.

### Step 3 — Choose Measurement Mode

> Select the measurement mode depending on the inspection method.

1. Tap the **Tape** tab to enter actual measurements manually (e.g., `32 1/2`).
2. Tap the **Deviation** tab to record deviation from the expected specification.
3. The selected tab will be highlighted to confirm your choice.

| Mode | Description |
|---|---|
| 📏 **Tape** | Enter actual measurements manually |
| 📋 **Deviation** | Record deviation from expected spec |

### Step 5 — Choose Spec Type

> Select the spec type to load the correct measurement points.

1. Tap the **spec dropdown** below the line number.
2. Choose one of the available spec types:

| Option | Description |
|---|---|
| 📋 **Production Main Spec** | Load only the main specification points |
| 📋 **All Spec** | Load all specification points for the style |

3. The measurement table will populate based on your selection.

### Step 6 — Measure the Garment

> Use a measuring tape to take the actual measurements of the garment and enter them into the system.

1. Lay the garment flat on the table.
2. Use the **measuring tape** to measure each **Point of Measurement** listed in the table.
3. Enter the **actual measurement value** into the corresponding **Actual** field on the tablet.
4. Repeat for each measurement point until all fields are filled.

### Step 7 — Enter Actual Measurements

> Input the measured values into the **Actual** field for each measurement point.

1. For each **Point of Measurement**, tap the **Actual** input field.
2. Enter the measured value (e.g., `22 5/16`, `19 5/8`).
3. Tap the 🗑️ **delete icon** to clear a value if needed.
4. Leave the field as `—` if the measurement point has not been measured yet.
5. Repeat until all points are filled.

> ⚠️ Ensure values are entered in the correct format (e.g., `32 1/2`).

### Step 8 — Record Defect

> If a defect is found on the garment, tap the **DEFECT** button to record it.

1. Tap the **DEFECT** button at the top of the screen.
2. A **QC Sketch** screen will open showing the garment diagram.
3. Tap the **defect location** on the garment sketch to mark it.
4. Tap the **📷 Camera icon** to take a photo of the defective area.
5. Select the **Defect type** from the dropdown list (e.g., `Uneven Stitches`, `Dirty Stain`, `Cut Hole`, `Pullness`).
6. Select the **Cause** (e.g., `Human Issue`).
7. Tap **💾 Save** to confirm the defect record.
8. The system will show the result as **✖ Fail** if defects are recorded.
9. All scanned defect records will appear in the **Defect Scan History** list.

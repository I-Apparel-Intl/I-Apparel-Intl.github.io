# 🏭 Step 3 — WIP Operation

**Handle work operations at each production station.**

---

<div align="center">
  <img src="/img/s3.png" width="60%" alt="WIP Operation" />
</div>

---

## 📋 Description

This feature is used to manage individual work operations at each station. It tracks what task is being done at each step of production.

---

## 🔄 How to Use

### 1️⃣ Step 1 — Open WIP Operation

- Tap **WIP Operation** from the home menu
- The dropdown ▼ will expand showing sub-options

---

### 2️⃣ Step 2 — Choose Operation Type

| Operation | Description |
|---|---|
| 🧵 Sewing Offline | គំរូ Offline — sewing tasks done without network |
| 🌐 Sewing Online | គំរូ Online — sewing tasks synced in real time |
| 📦 Packing | កាប៉ែកម៉ូ — packing finished products for shipment |

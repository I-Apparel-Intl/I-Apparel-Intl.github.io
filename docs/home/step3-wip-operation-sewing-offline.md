# 🧵 Sewing Offline

**Select and record sewing operations while working offline (without a live line connection).**

---

## 📋 Description

The **Sewing Offline** feature allows operators to log their sewing step and order assignment independently from the production line system. It is used when the operator is working offline or is not yet connected to an active sewing line.

---

## 🔄 How to Use

### 1️⃣ Step 1 — Open Sewing Offline

- From the **Home Menu**, tap **WIP Operation**
- The menu will expand to show sub-options
- Tap **Sewing Offline** (ដេរ Offline)

### 2️⃣ Step 2 — View the Entry Form

The **Sewing Offline — Select an Option** screen will open with the following fields:

    <img src="/img/h1.jpeg" width="50%" alt="Order No field highlighted" />
  </div>

| Field | Description |
|---|---|
| **Operator Name / Badge** | Auto-filled with the logged-in operator (e.g. Sam Rong - D010666) |
| **Order No** | Select the active production order (e.g. IA15996 // F6WP073543_CRI) |
| **Color** | Auto-filled based on selected order (e.g. DK GREY MELANGE) |
| **Line** | Select the production line (e.g. Line 0) |
| **Step** | Select the sewing operation step to perform |
| **Is Init** | Toggle to mark as initial entry |

### 7️⃣ Step 6 — Toggle Is Init (Optional)

- The **Is Init** toggle appears below the Step field
- Tap the circle to enable it ✅ if this is the **first entry at the start of the shift**
- Leave it off for regular mid-shift entries

<div align="center">
  <img src="/img/init.png" width="60%" alt="Is Init toggle" />
  <p><em>🔘 Tap Is Init to mark this as the initial shift entry</em></p>
</div>

> **When to use Is Init:**
> - At the **start of a new shift** before recording any production
> - When re-initializing an operator's assignment on a line
> - Leave **unchecked** for all normal mid-shift scans

---

---

### 6️⃣ Step 7 — Confirm or Close

- Review all selected fields
- Toggle **Is Init** ✅ if this is the initial entry for the shift
- Tap **✅ OK យល់ព្រម** to confirm and save the entry
- Tap **✖ CLOSE បិទ** to cancel and go back

<div align="center">
  <img src="/img/ok1.png" width="60%" alt="Sewing Offline Confirm Buttons" />
</div>

### 7️⃣ Step 8 — Scan Tickets / Record Quantity

After confirming in Step 6, the **Sewing Offline** active screen opens and is ready to record production.

#### 📊 Screen Overview

| Element | Description |
|---|---|
| **User** | Logged-in operator (e.g. Sam Rong) |
| **Line** | Selected production line (e.g. Line 0) |
| **Step** | Selected sewing operation (e.g. 1. SN TACK LABEL AT WEARER LEFT OR RIGHT SIDE S...) |
| **Total Qty** | Running count of recorded pieces (e.g. 0/0) |

---

#### 🎛️ Available Action Buttons

| Button | Khmer | Action |
|---|---|---|
| **Next Ticket** | សំបុត្របន្ទាប់ | Move to the next ticket in the queue |
| **Qty: 1** | បរិមាណ | Record quantity — default is 1 per scan |
| **Manual Entry** | កាចូលដោយដៃ | Enter quantity manually without scanning |
| **Rectify Qty** | កែសម្រួល Qty | Correct or adjust a previously recorded quantity |
| **Hide Menu** | លាក់មុីនុយ | Collapse the top menu to expand the scanning area |
| **Clear History** | — | Clear all recorded scan history |

---

#### 📷 Camera Access

If the camera turns off due to inactivity, the following message will appear:

> **"The camera has been turned off due to inactivity."**
> កាមេរ៉ាត្រូវបានផ្ដាច់ដោយសារតែការមិនសកម្មៗ

<div align="center">
  <img src="/img/n1.jpg" width="60%" alt="Camera Inactive Screen with ACCESS CAMERA button" />
</div>

- Tap the 🔴 **ACCESS CAMERA** button to reactivate the scanner and resume scanning tickets

---

#### ✅ Summary

| Action | Button | Result |
|---|---|---|
| Go to next ticket | **Next Ticket** | Advances to the next queued ticket |
| Record 1 piece | **Qty: 1** | Logs one unit for the current ticket |
| Enter qty manually | **Manual Entry** | Opens manual input field |
| Fix recorded qty | **Rectify Qty** | Allows correction of a previous entry |
| Collapse top menu | **Hide Menu** | Hides top bar to expand scan area |
| Reset history | **Clear History** | Clears all logged scan records |
| Reactivate camera | **ACCESS CAMERA** | Restarts the camera after inactivity timeout |

# 🧵 Sewing Online

**Select and record sewing operations while connected to a live production line.**

---

## 📋 Description

The **Sewing Online** feature allows operators to log their sewing step and order assignment while connected to an active production line system. It is used when the operator is working online and assigned to a live sewing line.

---

## 🔄 How to Use

### 1️⃣ Step 1 — Open Sewing Online

- From the **Home Menu**, tap **WIP Operation**
- The menu will expand to show sub-options
- Tap **Sewing Online** (ដេរ Online)

---

### 2️⃣ Step 2 — View the Entry Form

The **Sewing Online — Select an Option** screen will open with the following fields:

<div align="center">
  <img src="/img/o11.png" width="50%" alt="Sewing Online entry form" />
</div>

| Field | Description |
|---|---|
| **Order No** | Select the active production order (e.g. IA16264 // 1000133779) |
| **Color** | Auto-filled based on selected order (e.g. 17-4433TCX) |
| **Line** | Select the production line (e.g. Line 09) |
| **Step** | Select the sewing operation step to perform |
| **Is Init** | Toggle to mark as initial entry |

---

### 3️⃣ Step 3 — Select Order No

- Tap the **Order No** dropdown (highlighted in green) to expand the order list
- Tap the 🔴 **QR icon** on the right to scan a QR code instead of selecting manually
- The QR scanner will open — point your camera at the QR code to retrieve the order number and color automatically

### 7️⃣ Step 7 — Toggle Is Init (Optional)

- The **Is Init** toggle appears below the Step field
- Tap the circle to enable it ✅ if this is the **first entry at the start of the shift**
- Leave it off for regular mid-shift entries

<div align="center">
  <img src="/img/init.png" width="60%" alt="Is Init toggle" />
  <p><em>🔘 Tap Is Init to mark this as the initial shift entry</em></p>
</div>

> **When to use Is Init:**
> - At the **start of a new shift** before recording any production
> - When re-initializing an operator's assignment on a line
> - Leave **unchecked** for all normal mid-shift scans

---

### 8️⃣ Step 8 — Confirm or Close

- Review all selected fields
- Tap **✅ OK យល់ព្រម** to confirm and save the entry
- Tap **✖ CLOSE បិទ** to cancel and go back

#### 📊 Screen Overview

| Element | Description |
|---|---|
| **User** | Logged-in operator (e.g. Sam Rong) |
| **Line** | Selected production line (e.g. Line 09) |
| **Step** | Selected sewing operation (e.g. 5. LOADING 2BODY, 2SLEEVE...) |
| **Total Qty** | Running count of recorded pieces (e.g. 0/0) |

---

#### 🎛️ Available Action Buttons

| Button | Khmer | Action |
|---|---|---|
| **Next Ticket** | សំបុត្របន្ទាប់ | Move to the next ticket in the queue |
| **Qty: 1** | បរិមាណ | Record quantity — default is 1 per scan |
| **Manual Entry** | កាចូលដោយដៃ | Enter quantity manually without scanning |
| **Rectify Qty** | កែសម្រួល Qty | Correct or adjust a previously recorded quantity |
| **Hide Menu** | លាក់មុីនុយ | Collapse the top menu to expand the scanning area |
| **Clear History** | — | Clear all recorded scan history |

#### ✅ Summary

| Action | Button | Result |
|---|---|---|
| Open Sewing Online | WIP Operation → Sewing Online | Opens the entry form |
| Select order manually | Order No dropdown | Shows order list to choose from |
| Select order by scan | 🔴 QR icon next to Order No | Opens camera to scan QR code |
| Color auto-fill | — | Filled automatically after order selection |
| Select production line | Line dropdown | Choose Line 09, etc. |
| Select sewing step | Step dropdown or 🔵 QR icon | Choose or scan the sewing operation |
| Mark as initial entry | Is Init toggle | Flags record as shift-start entry |
| Save the entry | **OK យល់ព្រម** | Confirms and records the sewing operation |
| Cancel | **CLOSE បិទ** | Discards changes and returns to menu |
| Go to next ticket | **Next Ticket** | Advances to the next queued ticket |
| Record 1 piece | **Qty: 1** | Logs one unit for the current ticket |
| Enter qty manually | **Manual Entry** | Opens manual input field |
| Fix recorded qty | **Rectify Qty** | Allows correction of a previous entry |
| Collapse top menu | **Hide Menu** | Hides top bar to expand scan area |
| Reset history | **Clear History** | Clears all logged scan records |
| Reactivate camera | **ACCESS CAMERA** | Restarts the camera after inactivity timeout |

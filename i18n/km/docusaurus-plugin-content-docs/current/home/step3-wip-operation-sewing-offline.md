# рҹ§ө бһҠбҹҒбһҡ Offline

**бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһ“бһ·бһ„бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹҒбһҡ бһҒбһҺбҹҲбһ–бҹҒбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹ’бһҡбҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ¶бһүбһ•бҹ’бһ‘бһ¶бһӣбҹӢ)бҹ”**

---

## рҹ“Ӣ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ–бһҺбҹҢбһ“бһ¶

бһҳбһ»бһҒбһ„бһ¶бһҡ **бһҠбҹҒбһҡ Offline** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҮбҹҶбһ бһ¶бһ“бһҠбҹҒбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ¶бһҸбҹӢбһҸбһ¶бҹҶбһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү бһҠбҹ„бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹ’бһҡбҹ…бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ¬бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹҒбһҡбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

---

## рҹ”„ бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ

### 1пёҸвғЈ бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 1 вҖ” бһ”бһҫбһҖ бһҠбҹҒбһҡ Offline

- бһ–бһё **бһҳбҹүбһәбһ“бһ»бһҷбһҳбҹҒ** бһ…бһ»бһ… **WIP Operation**
- бһҳбҹүбһәбһ“бһ»бһҷбһ“бһ№бһ„бһ–бһ„бҹ’бһҡбһёбһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҮбһҳбҹ’бһҡбһҫбһҹбһҡбһ„
- бһ…бһ»бһ… **Sewing Offline** (бһҠбҹҒбһҡ Offline)

### 2пёҸвғЈ бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 2 вҖ” бһҳбһҫбһӣбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бһүбҹ’бһ…бһјбһӣ

бһўбҹҒбһҖбҹ’бһҡбһ„бҹӢ **бһҠбҹҒбһҡ Offline вҖ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбһҳбҹ’бһҡбһҫбһҹ** бһ“бһ№бһ„бһ”бһҫбһҖбһҠбҹ„бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһўбһ”бҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

|---|---|
| **бһҲбҹ’бһҳбҹ„бҹҮбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡ / бһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ** | бһ”бҹҶбһ–бҹҒбһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһјбһӣ (бһ§. Sam Rong - D010666) |
| **бһӣбҹҒбһҒбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү** | бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһүбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (бһ§. IA15996 // F6WP073543_CRI) |
| **бһ–бһҺбҹҢ** | бһ”бҹҶбһ–бҹҒбһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһүбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ (бһ§. DK GREY MELANGE) |
| **бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ** | бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ (бһ§. Line 0) |
| **бһҮбҹҶбһ бһ¶бһ“** | бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбҹҶбһ бһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹҒбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸ |
| **Is Init** | бһ”бһҫбһҖ/бһ”бһ·бһ‘бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹҶбһ”бһјбһ„ |

### 3пёҸвғЈ бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 3 вҖ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӣбҹҒбһҒбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү

- бһ…бһ»бһ… **бһҳбҹүбһәбһ“бһ»бһҷбһ‘бһҳбҹ’бһӣбһ¶бһҖбҹӢбһӣбҹҒбһҒбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү** (бһ”бһ“бҹ’бһӣбһ·бһ…бһ–бһҺбҹҢбһ”бҹғбһҸбһ„) бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖбһ”бһүбҹ’бһҮбһё
- бһ…бһ»бһ… рҹ”ҙ **бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ QR** бһ“бҹ…бһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһҖбҹҒбһ“ QR Code бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҠбҹ„бһҷбһҠбҹғ
- бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбҹ’бһҖбҹҒбһ“ QR бһ“бһ№бһ„бһ”бһҫбһҖ вҖ” бһ”бһ„бҹ’бһңбҹӮбһҡбһҖбһ¶бһҳбҹҒбһҡбҹүбһ¶бһ‘бҹ… QR Code бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһӣбҹҒбһҒбһ”бһүбҹ’бһҮбһ¶бһ‘бһ·бһү бһ“бһ·бһ„бһ–бһҺбҹҢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

### 4пёҸвғЈ бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 4 вҖ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бһҺбҹҢ

### 7пёҸвғЈ бһҮбҹҶбһ бһ¶бһ“бһ‘бһё 6 вҖ” бһ”бһҫбһҖ Is Init (бһҮбһҳбҹ’бһҡбһҫбһҹ)

- бһ”бҹ’бһҸбһјбһҡ **Is Init** бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бҹ’бһҡбһўбһ”бҹӢбһҮбҹҶбһ бһ¶бһ“
- бһ…бһ»бһ…бһҡбһ„бҹ’бһңбһ„бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖ вң… бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ“бҹҒбҹҮбһҮбһ¶ **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹҶбһ”бһјбһ„бһ“бҹ…бһҠбһҫбһҳбһңбҹҒбһ“**
- бһ‘бһ»бһҖбһұбҹ’бһҷбһ”бһ·бһ‘бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ’бһҳбҹ’бһҳбһҸбһ¶бһҖбҹ’бһ“бһ»бһ„бһңбҹҒбһ“

> - бһ‘бһ»бһҖ **бһҳбһ·бһ“бһ”бһ¶бһ“бһӮбһјбһҹбһҹбһүбҹ’бһүбһ¶** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“бһ’бһҳбҹ’бһҳбһҸбһ¶бһҖбҹ’бһ“бһ»бһ„бһңбҹҒбһ“

# 🔍 ការវាស់វែងដំណាក់កាលបញ្ចប់

**វាស់វែង និងផ្ទៀងផ្ទាត់ផលិតផលដែលបានបញ្ចប់មុនការអនុម័តចុងក្រោយ។**

## 📋 សេចក្តីពិពណ៌នា

មុខងារនេះត្រូវបានប្រើដើម្បីវាស់វែង និងត្រួតពិនិត្យផលិតផលដែលបានបញ្ចប់ ដើម្បីធានាថាពួកវាបំពេញតាមស្តង់ដារគុណភាព មុនពេលបន្តទៅដំណាក់កាលបន្ទាប់។

## 🪜 ជំហាន

### ជំហានទី ១ — ស្កេន ID កម្មករ

> មុនពេលបន្ត កម្មករត្រូវតែត្រូវបានកំណត់អត្តសញ្ញាណ។

1. ចុចលើ **📷 រូបតំណាងកាមេរ៉ា** ឬ **🔍 រូបតំណាង QR** នៅខាងឆ្វេងខាងលើ ដើម្បីបើកម៉ាស៊ីនស្កេន។
2. **ស្កេន QR Code កម្មករ** ដើម្បីទាញយកព័ត៌មានកម្មករ។
3. នៅពេលស្កេនរួច ឈ្មោះ និង ID កម្មករនឹងបង្ហាញ (ឧ. `NHANH SOKHAI - D010966`)។

### ជំហានទី ២ — ជ្រើសរើសខ្សែ

> ជ្រើសរើសខ្សែផលិតកម្មមុនពេលបន្តការវាស់វែង។

1. បន្ទាប់ពី ID កម្មករត្រូវបានទាញយក **ប្រអប់ dropdown** នឹងបង្ហាញខាងក្រោមឈ្មោះកម្មករ។
2. ចុចលើ dropdown ហើយ **ជ្រើសរើសលេខខ្សែ** (ឧ. `02`)។
3. ខ្សែដែលបានជ្រើសរើសនឹងត្រូវបានបញ្ជាក់ និងបង្ហាញនៅក្នុងប្រអប់។

### ជំហានទី ៣ — ជ្រើសរើសរបៀបវាស់វែង

> ជ្រើសរើសរបៀបវាស់វែងអាស្រ័យលើវិធីសាស្ត្រត្រួតពិនិត្យ។

1. ចុចផ្ទាំង **Tape** ដើម្បីបញ្ចូលការវាស់វែងជាក់ស្តែងដោយដៃ (ឧ. `32 1/2`)។
2. ចុចផ្ទាំង **Deviation** ដើម្បីកត់ត្រាការបន្លាស់ប្តូរពីការបញ្ជាក់ដែលរំពឹងទុក។
3. ផ្ទាំងដែលបានជ្រើសរើសនឹងត្រូវបានបន្លិចដើម្បីបញ្ជាក់ជម្រើសរបស់អ្នក។

| របៀប | ការពិពណ៌នា |
|---|---|
| 📏 **Tape** | បញ្ចូលការវាស់វែងជាក់ស្តែងដោយដៃ |
| 📋 **Deviation** | កត់ត្រាការបន្លាស់ប្តូរពីការបញ្ជាក់ |

### ជំហានទី ៤ — ប្រើរូបតំណាងនៅរបារខាងលើ

> ប្រើរូបតំណាងនៅរបារខាងឆ្វេងខាងលើសម្រាប់សកម្មភាពបន្ថែម។

1. ចុចលើ **📷 រូបតំណាងកាមេរ៉ា** ដើម្បីស្កេន QR Code កម្មករដោយប្រើកាមេរ៉ា។
2. ចុចលើ **🔍 រូបតំណាង QR Scan** ដើម្បីស្កេនដោយប្រើ Google QR scanner។
3. ចុចលើ **🕐 រូបតំណាង History** ដើម្បីមើលកំណត់ត្រាការវាស់វែងមុនៗ។

### ជំហានទី ៥ — ជ្រើសរើសប្រភេទ Spec

> ជ្រើសរើសប្រភេទ spec ដើម្បីទាញយកចំណុចវាស់វែងត្រឹមត្រូវ។

1. ចុចលើ **ប្រអប់ dropdown spec** ខាងក្រោមលេខខ្សែ។
2. ជ្រើសរើសប្រភេទ spec ដែលមាន៖

| ជម្រើស | ការពិពណ៌នា |
|---|---|
| 📋 **Production Main Spec** | ទាញយកតែចំណុចការបញ្ជាក់សំខាន់ |
| 📋 **All Spec** | ទាញយកចំណុចការបញ្ជាក់ទាំងអស់សម្រាប់រចនាប័ទ្ម |

3. តារាងវាស់វែងនឹងបំពេញដោយផ្អែកលើការជ្រើសរើសរបស់អ្នក។

### ជំហានទី ៦ — វាស់វែងសម្លៀកបំពាក់

> ប្រើខ្សែបង្ហួរដើម្បីវាស់វែងជាក់ស្តែងនូវសម្លៀកបំពាក់ ហើយបញ្ចូលវាទៅក្នុងប្រព័ន្ធ។

1. រៀបសម្លៀកបំពាក់ឱ្យរាបស្មើលើតុ។
2. ប្រើ **ខ្សែបង្ហួរ** ដើម្បីវាស់វែង **ចំណុចវាស់វែង** នីមួយៗដែលបានរាយក្នុងតារាង។
3. បញ្ចូល **តម្លៃការវាស់វែងជាក់ស្តែង** ទៅក្នុងប្រអប់ **Actual** ដែលត្រូវគ្នានៅលើថេប្លេត។
4. ធ្វើម្តងទៀតសម្រាប់ចំណុចវាស់វែងនីមួយៗរហូតដល់ប្រអប់ទាំងអស់ត្រូវបានបំពេញ។

### ជំហានទី ៧ — បញ្ចូលការវាស់វែងជាក់ស្តែង

> បញ្ចូលតម្លៃដែលបានវាស់វែងទៅក្នុងប្រអប់ **Actual** សម្រាប់ចំណុចវាស់វែងនីមួយៗ។

1. សម្រាប់ **ចំណុចវាស់វែង** នីមួយៗ ចុចលើប្រអប់បញ្ចូល **Actual**។
2. បញ្ចូលតម្លៃដែលបានវាស់វែង (ឧ. `22 5/16`, `19 5/8`)។
3. ចុចលើ 🗑️ **រូបតំណាងលុប** ដើម្បីលុបតម្លៃប្រសិនបើចាំបាច់។
4. ទុកប្រអប់ជា `—` ប្រសិនបើចំណុចវាស់វែងមិនទាន់បានវាស់វែងនៅឡើយ។
5. ធ្វើម្តងទៀតរហូតដល់ចំណុចទាំងអស់ត្រូវបានបំពេញ។

> ⚠️ ត្រូវប្រាកដថាតម្លៃត្រូវបានបញ្ចូលក្នុងទម្រង់ត្រឹមត្រូវ (ឧ. `32 1/2`)។

### ជំហានទី ៨ — កត់ត្រាចំណុចខ្វះខាត

> ប្រសិនបើរកឃើញចំណុចខ្វះខាតនៅលើសម្លៀកបំពាក់ ចុចប៊ូតុង **DEFECT** ដើម្បីកត់ត្រា។

1. ចុចប៊ូតុង **DEFECT** នៅខាងលើអេក្រង់។
2. អេក្រង់ **QC Sketch** នឹងបើកបង្ហាញដ្យាក្រាមសម្លៀកបំពាក់។
3. ចុចលើ **ទីតាំងចំណុចខ្វះខាត** នៅលើដ្យាក្រាមសម្លៀកបំពាក់ដើម្បីសម្គាល់វា។
4. ចុចលើ **📷 រូបតំណាងកាមេរ៉ា** ដើម្បីថតរូបតំបន់ដែលមានចំណុចខ្វះខាត។
5. ជ្រើសរើស **ប្រភេទចំណុចខ្វះខាត** ពី dropdown (ឧ. `Uneven Stitches`, `Dirty Stain`, `Cut Hole`, `Pullness`)។
6. ជ្រើសរើស **មូលហេតុ** (ឧ. `Human Issue`)។
7. ចុច **💾 Save** ដើម្បីបញ្ជាក់កំណត់ត្រាចំណុចខ្វះខាត។
8. ប្រព័ន្ធនឹងបង្ហាញលទ្ធផលជា **✖ Fail** ប្រសិនបើមានចំណុចខ្វះខាតត្រូវបានកត់ត្រា។
9. កំណត់ត្រាចំណុចខ្វះខាតដែលបានស្កេនទាំងអស់នឹងបង្ហាញនៅក្នុងបញ្ជី **Defect Scan History**។

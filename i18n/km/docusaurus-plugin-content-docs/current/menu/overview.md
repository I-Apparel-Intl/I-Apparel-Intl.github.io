# рҹ“– бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

## бһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶ LineGuru?

**LineGuru** бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһҠбҹҸбһ‘бҹҶбһ“бһҫбһ”бһҳбһҪбһҷ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһҖбһ¶бһҸбҹӢбһҠбҹҒбһҡ бһҠбҹ„бһҷбһҮбһҪбһҷбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбһӣбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһҸбһ¶бҹҶбһ„бһ–бһёбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҖбһ¶бһҸбҹӢ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”

бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбһ бһҖбһ¶бһҡбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһұбҹ’бһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ“бҹғбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҹбһҳбҹ’бһӣбҹҖбһҖбһ”бҹҶбһ–бһ¶бһҖбҹӢбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ—бһ¶бһ–бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

бһҮбһ¶бһҳбһҪбһҷ LineGuru бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһ•бһӣбһ·бһҸбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһ”бһ¶бһ“бһ“бһјбһңбһ‘бһ·бһ“бҹ’бһ“бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹбһҶбҹ’бһӮбһ„ бһ“бһ·бһ„бһӣбһҫбһҖбһҖбһҳбҹ’бһ–бһҹбҹӢбһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҖбһ¶бһҡбһ„бһ¶бһҡбһ‘бһјбһ‘бҹ…бһұбҹ’бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бҹ”
